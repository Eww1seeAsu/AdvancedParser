# 🚀 AdvancedParser

**Асинхронный парсер с обходом капчи (hCaptcha, reCaptcha), веб-интерфейсом и автоповтором при ошибках.**

---

## 📌 Возможности

- 🛡️ Обход капчи через Capsolver
- 🔄 Работа через прокси с ротацией
- 📦 Парсинг товаров с пагинацией
- 💾 Сохранение данных в SQLite и CSV
- 🌐 Веб-интерфейс на FastAPI
- 📝 Логирование всех действий
- 🔁 Автоматическое повторение запросов при ошибках

---

## ⚙️ Установка и запуск

```bash
git clone https://github.com/твой_ник/AdvancedParser.git
cd AdvancedParser
pip install -r requirements.txt
playwright install
python main.py
